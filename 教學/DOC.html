[密技] 用 7-Zip 自製免安裝軟件，走到哪用到哪

3月 27, 2013 科技產品 , 密技 , 程式開發 3 留言


   今天在整理文稿的時候，發現一篇三年前寫下的文章：把要安裝的軟件綠色化，變成Portable !，裡面記載了自製 Portable 軟件的方法，可是寫得有點亂。於是我決定再寫一次，並優化了部份程序，希望這密技能為對免安裝軟件有高度需求的網友帶來方便吧。

1. 準備工作

  • 安裝 7-Zip 及 Universal Extractor
  • 然後安裝 7-Zip SFX Maker，記住安裝路徑

 

2. 自製免安裝軟件

  • 打開 Universal Extractor

  • 在 “Archive/Installer to extract” 裡開啓程式的 .msi 安裝檔

  • 然後按 OK

  • 若果出現下列警告無需理會，點 OK 即可

  • 按 OK

  • Universal Extractor 會開始解壓 MSI，並把程式檔案抽取出來放在新資料夾上

 

  • .exe 是程式的主要執行檔

  • 全選資料夾內的檔案，按右鍵 –> Add to archive…

  • 把 Archive format 設定為 7z，按 OK

  • 接著到 7-ZIP SFX Maker 的安裝路徑，打開 7-ZIP SFX Maker.exe

  • 把剛壓縮的 7z 檔拖放 SFX Maker 上

  • 到 Tasks –> 按右面的 + –> 選 Run Program –> 按 OK

  • 直接在 %%T\ 後面輸入 exe 執行檔的名稱

  • 在 Dialogs 的 Extract Path 裡取消勾選 Allow user to change path

  • 然後再到 General 勾選 Extract to temporary folder，並自訂免安裝包的名稱，設定完成後按 Make SFX 即可

  • 完成後，資料夾會出現 xxxxx.sfx.exe，這就是你自製的免安裝程式嚕

  • 試試把它放到 USB 記憶體裡開啓，看看程式能否正常運行

這是我在這教學中自製的 Portable 軟件，有興趣的網友可以下載試試：https://mega.co.nz/#!3YkC2b6R!N-8iTUNyKSy4pZQI3pZD20nhTIQPwxsq_VJXz88FvAI

如果各位對於用 7-ZIP 自製免安裝軟件有甚麼問題或意見，歡迎留言與我討論一下喔:-)
